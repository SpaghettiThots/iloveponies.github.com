Functional Programming Workshop

A course on functional programming with Clojure.

Clojure is a young Lispish functional programming language on the JVM platform. Because it runs on the JVM, all Clojure programs can use all the standard and third-party Java libraries freely.

Clojure is suitable for small and large programs. It offers tools for many tasks that are harder with other languages:

  • Immutable data structures, agents and software transactional memory ease development of robust concurrent systems.
  • Macros allow building more expressive domain specific languages.
  • Typing hints and localized mutability let you gain performance where it is needed.
